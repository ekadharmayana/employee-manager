# Employee Management API
A professional backend system for managing employees. Developed with Java 21, Spring Boot 3, and PostgreSQL.

## Technologies
* **Java 21**
* **Spring Boot 3**
* **PostgreSQL** (Database)
* **Docker & Docker Compose** (Containerization)
* **GitHub Actions** (CI/CD)

## Requirements
* JDK 21
* Docker Desktop
